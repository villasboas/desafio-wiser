# Como rodar

Clone este repositório

```bash
    git clone https://github.com/villasboas/desafio-wiser.git
    cd desafio-wiser
```

Instale as dependencias do projeto

```bash
yarn install
```

Rode o projeto localmento

```
yarn dev
```

O projeto ficará disponível em http://localhost:3000
